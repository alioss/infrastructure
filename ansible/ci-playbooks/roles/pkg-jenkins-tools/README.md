Package: jenkins-tools
==================

Install tools needed by Jenkins, including

- Shells (bash)
- Archival tools (tar, gzip, bzip2, lzma, xz)
- Download and upload tools (curl, wget, rsync)
- Miscellaneous tools (fakeroot)

- FreeBSD (install tools)

Requirements
------------

None.
